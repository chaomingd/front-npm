## API

### 参数
| 参数    | 描述                                                    | 类型                              | 必填 | 默认值                                |
| ------- | ------------------------------------------------------- | --------------------------------- | ---- | ------------------------------------- |
| `arr`   | 要转换的数组                                            | `T[]`                             | 是   | 无                                    |
| `getKey` | 用于生成对象键的函数                                    | `(item: T) => string`             | 否   | `(item) => item`                      |
| `getValue` | 用于生成对象值的函数                                  | `(item: T) => any`                | 否   | `(item: T) => item`                   |

### 返回值
| 返回值 | 描述                        | 类型                                      |
| ------ | --------------------------- | ----------------------------------------- |
| `map`  | 转换后的对象，其键和值分别由 `getKey` 和 `getValue` 函数生成 | `Record<string, ReturnType<GetValue>>`    |
