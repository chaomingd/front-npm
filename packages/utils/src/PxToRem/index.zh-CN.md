# pxtorem
将像素值转换为 rem 单位值，并支持设置小数点精度。

<code src="./demos/basic.tsx"></code>


## API

### 参数
| 参数       | 描述                           | 类型      | 必填 | 默认值 |
| ---------- | ------------------------------ | --------- | ---- | ------ |
| `px`       | 要转换的像素值                 | `number`  | 是   | 无     |
| `precision` | 结果中小数点后的位数           | `number`  | 否   | `5`    |


### 返回值
| 返回值     | 描述                                                 | 类型      |
| ---------- | ---------------------------------------------------- | --------- |
| `string`   | 转换后的 rem 值（如果输入的 px 为 0，则返回 `undefined`） | `string \| undefined` |
