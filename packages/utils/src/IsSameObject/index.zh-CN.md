# isSameObject
比较两个对象是否相同。它可以选择性地只比较指定的键，或者比较两个对象的所有键。

## API
### 参数
| 参数    | 描述                                  | 类型                     | 必填 | 默认值  |
| ------- | ------------------------------------- | ------------------------ | ---- | ------- |
| `obj1`  | 第一个要比较的对象                    | `Record<string, any>`    | 是   | 无      |
| `obj2`  | 第二个要比较的对象                    | `Record<string, any>`    | 是   | 无      |
| `keys`  | 选择性比较的键数组，只比较这些键对应的值 | `string[]`               | 否   | `keys1` |


### 返回值
| 返回值 | 描述                                  | 类型      |
| ------ | ------------------------------------- | --------- |
| `true` | 如果两个对象相同，返回 `true`         | `boolean` |
| `false` | 如果两个对象不同，返回 `false`       | `boolean` |
