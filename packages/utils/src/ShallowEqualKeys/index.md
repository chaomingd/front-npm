## API
### 参数
| 参数   | 描述                                           | 类型           | 必填 | 默认值 |
| ------ | ---------------------------------------------- | -------------- | ---- | ------ |
| `obj1` | 第一个要比较的对象                           | `object`       | 是   | 无     |
| `obj2` | 第二个要比较的对象                           | `object`       | 是   | 无     |
| `keys` | 选择性比较的键数组，只比较这些键对应的值     | `string[]`     | 否   | `undefined` |


### 返回值
| 返回值  | 描述                                                  | 类型    |
| ------- | ----------------------------------------------------- | ------- |
| `boolean` | 如果指定的键在两个对象中具有相同的值，返回 `true`；否则返回 `false`。 | `boolean` |
