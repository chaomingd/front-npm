## API
### 参数
| 参数     | 描述                                                        | 类型                                | 必填 | 默认值 |
| -------- | ----------------------------------------------------------- | ----------------------------------- | ---- | ------ |
| `type`   | 事件的名称，用于指定要监听或触发的事件                     | `TEventName`（`string` 类型的键）   | 是   | 无     |
| `fn`     | 事件处理函数，当事件触发时调用该函数                        | `IFunction`                         | 是   | 无     |
| `args`   | 触发事件时传递给事件处理函数的参数                         | `any`                               | 否   | 无     |
| `listeners` | 事件处理函数数组，用于存储所有监听该事件的函数           | `IFunction[]` 或 `undefined`        | 否   | 无     |

### 返回值
| 返回值   | 描述                                      | 类型          |
| -------- | ----------------------------------------- | ------------- |
| `Function` | `on` 方法返回一个函数，用于取消事件监听器   | `() => void`   |
| `boolean` | `emit` 方法返回 `true` 如果事件被触发并且有监听器，否则返回 `false` | `boolean`     |
